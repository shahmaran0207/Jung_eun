1. 파이썬에서 과거 데이터를 통해 향후 데이터 예측 
2. spring의 각 페이지에서 입려값을 controller로 받아옴 
3. servic로 받은 값을 넘겨 계산할 준비
4. 파이썬에 저장된 여러 예측값들을 spring  service로 가져옴(어케할지 고민)
5. 가져온 값을 통해 계산
ex) 소비자가 강원도에서 3일동안 숙박, 3끼 먹을거라 입력 
- > 파이썬에서 가져온  변수 중 23년 강원도에서 숙박(200원), 식사(300원) 데이터가 있음
->  service 페이지에서 계산: 200 x 3 + 300 x 3 = 1,500원 RESULT 페이지에 출력
