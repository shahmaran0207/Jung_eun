필수!
ord(문자) -> 아스키 코드 숫자
chr(숫자) -> 아스키 코드 문자
bin(숫자) -> 2진수 [2:] 해줘야 함
gcd() -> 괄호 안의 숫자들의 최대공약수 구해주는 함수
math.lcm() -> 최소공배수 구하는 함수

(x * y) / gcd(x,y) -> 최소공배수

isinstance(n,k): n이라는 숫자가 k라는 형태의 숫자 타입인지 확인

올림보장 -> (s+k-1)//k

리스트 뒤집기
[::-1]

variable.startwith('n'): varijable 문자열이 n으로 시작하는지 확인
variable.endwith('n'): varijable 문자열이 n으로 끝나는지 확인
